# ROASTBROS Coffee Website

This is a static coffee shop website created for the Web Design course.

## Structure
- index.html (Homepage)
- about.html (About / Corporate)
- menu.html (Menu)
- contact.html (Contact)
- styles.css (Global stylesheet)
- images/ (All images)

## Tools
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- Google Fonts

## How to View
Open index.html or visit the GitHub Pages link.
